# Test Automation Framework (In Progress)

This project is a personal automation testing framework designed to simulate real-world enterprise testing architecture.

The goal of the project is to build a scalable, maintainable, and extensible UI/API automation framework using industry best practices. It is intended as a learning and experimentation environment focused on clean architecture, modular design, and long-term maintainability.

---

## Features

- Page Object Model (POM) architecture
- UI automation layer based on Selenium
- API testing layer using RestAssured
- Modular and extensible project structure
- CI-ready design
- Reporting integration (planned)

---

## Tech Stack

- Java
- Selenium
- RestAssured
- JUnit
- Spring
- Maven
- Allure
- Jenkins
